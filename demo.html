INTRODUCCIÓN

El alojamiento web u hospedaje web es el servicio que provee a los usuarios de Internet un espacio de almacenamiento en línea, también conocido como webspace, que permite publicar todo el contenido relacionado con un sitio web

QUE VAMOS A VER EN ESTA PRESENTACIÓN

En esta presentación vamos a ver las ventajas e inconvenientes de los tipos de alojamientos web y los aspectos que tenemos que tener en cuenta para poder publicar una pagina web

HOSTING GRATUITO

VENTAJAS: Hay sitios profesionales para alojar tu sitio sencillo de manera gratuita y tambien puedes guardar archivos en ubicaciones diferentes para ocupar menos espacio

DESVENTAJAS: Es posible que no puedas monetizar tu sitio web ademas de que el espacio es limitado

HOSTING COMPARTIDO

VENTAJAS: El alojamiento web es muy económico, el proveedor del hosting se encargará de la administracion y el mantenimiento del servidor, la gestion es mas sencilla, dispone tambien un servicio de atencion al cliente y certificados SSL

DESVENTAJAS: La personalizacion tiene limites, tambien las cargas y procesos son mas lentos y pueden contener problemas de seguridad

HOSTING PARA REVENDEDORES

VENTAJAS: Los precios son muy bajos y ademas te permite revender los recursos

DESVENTAJAS: El espacio yu la transferencia son limitados, tiene independencia de IP's

HOSTING VPS

VENTAJAS: Los recursos son dedicados con un procesador y ram dedicada a ese host, tiene flexibilidad y la privacidad es mayor

DESVENTAJAS: El costo puede ser mas elevado, el sistema de datos requiere mas preparacion, la seguridad puede encontrarse en vulnerabilidad y hay limitaciones

HOSTING DEDICADO

VENTAJAS: Tiene mas control del servidor, Dispone de todos los recursos e incrementa la seguridad

DESVENTAJAS: El precio es mas alto y son mas dificiles de gestionar

CLOUD HOSTING

VENTAJAS: Escalabilidad y flexibilidad, optimizacion de costes, mas disponibilidad, mejora de la seguridad y mas facil de usar

DESVENTAJAS: Un proveedor de confianza, seguridad, tiempo de inactividad y tiene mas accesibilidad
